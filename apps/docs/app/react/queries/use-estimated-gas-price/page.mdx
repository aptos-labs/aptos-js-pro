# useEstimatedGasPrice

Hook for fetching the current estimated gas price on the network.

## Usage

```tsx
import { useEstimatedGasPrice } from "@aptos-labs/react";

export default function App() {
  const { data, isLoading } = useEstimatedGasPrice();

  if (isLoading) return <div>Loading...</div>;

  return (
    <div>
      <p>Estimated Gas Price: {data?.gasEstimate}</p>
      <p>Deprioritized Gas Estimate: {data?.deprioritizedGasEstimate}</p>
      <p>Prioritized Gas Estimate: {data?.prioritizedGasEstimate}</p>
    </div>
  );
}
```
