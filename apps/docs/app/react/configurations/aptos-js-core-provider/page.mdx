# AptosJSCoreProvider

The `AptosJSCoreProvider` is a React context provider that provides a `AptosJSProClient` instance to the application.

## Usage

```tsx
import { AptosJSCoreProvider } from "@aptos-labs/react";

export default function App() {
  const core = new AptosJSProClient({
    /* ... */
  });

  return (
    <AptosJSCoreProvider core={core}>
      {/* Your app components */}
    </AptosJSCoreProvider>
  );
}
```
