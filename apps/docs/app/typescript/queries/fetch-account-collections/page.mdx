# fetchAccountCollections

Fetching collections for a specific account, with support for pagination.

## Usage

```ts
import { AptosJSProClient } from "@aptos-labs/js-pro";

const client = new AptosJSProClient({
  /* ... */
});

const { nextCursor, prevCursor, collections } =
  await client.fetchAccountCollections({
    address:
      "0x926ff619e85bdb5b02680f27bea5ea3302c71f86be4b16fa0d63df213add927a",
  });
```
