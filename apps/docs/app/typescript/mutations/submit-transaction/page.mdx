# submitTransaction

This function is used to submit signed transactions to the Aptos blockchain.

## Usage

```ts
import { AptosJSProClient } from "@aptos-labs/js-pro";

const client = new AptosJSProClient({
  /* ... */
});

const transaction = await client.buildTransaction({
  /* ... */
});

const signedTransaction = await client.signTransaction({ transaction });

const transactionResponse = await client.submitTransaction({
  senderAuthenticator: signedTransaction.authenticator,
  transaction: signedTransaction,
});
```
